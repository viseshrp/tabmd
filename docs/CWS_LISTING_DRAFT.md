# Chrome Web Store Listing Draft

## Store Listing Specs

- **Name**: TabMD - Markdown Editor
- **Short Description**: Turn every new tab into a distraction-free Markdown editor.
- **Category**: Productivity

## Detailed Description

TabMD replaces your default new tab page with a beautiful, lightweight, and incredibly fast Markdown editor.

We believe that your thoughts should be captured instantly. That's why every time you open a new tab, you are greeted with a fresh note editor.

**Key Features:**
*   **Instant Access**: No loading screens or spinners. Just start typing on a new tab.
*   **True Markdown Support**: Complete GitHub Favored Markdown (GFM) compatibility. Write tables, task lists, and perfectly formatted code blocks with syntax highlighting.
*   **100% Private and Local**: Your notes are automatically saved directly to an offline, unlimited database in your browser. We have zero tracking, analytics, or cloud syncing.
*   **Effortless Management**: Seamlessly browse, search, and delete your notes from the built-in popup and list view.
*   **Save-on-Blur**: Your notes are instantly saved without any manual clicking or saving buttons.
*   **Theme Integration**: Sync your editor interface to match your operating system, or manually choose between Light or Dark modes.

TabMD is free and open-source. Replace your boring empty tab with the notes you need. Start writing with TabMD.
