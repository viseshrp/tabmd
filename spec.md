# Product Spec

## Goal
Provide a maintainable WXT browser-extension starter with a clear project organization, delivery pipeline, and generic feature placeholders.

## Scope
- Browser action opens a dashboard page.
- Dashboard shows starter collections and resource cards.
- Options page manages generic settings.
- Drive integration remains a placeholder surface for future work.

## Non-goals
- Shipping product-specific behavior in the scaffold.
- Shipping production-specific business logic.
